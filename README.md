## Практическая работа 3
Данная работа посвящена работе с указателями, динамической памятью и массивами в языке C. В проекте реализованы 4 задачи, каждая из которых выполняет различные операции с памятью и указателями.

## Задачи

1. **Указатели и операции с ними**: Увеличение и уменьшение значений переменных с помощью указателей.
2. **Динамическая память**: Работа с динамически выделенной памятью для вещественных переменных.
3. **Массивы и указатели**: Создание динамического массива, нахождение отрицательных элементов и вычисление их среднего значения.
4. **Корни квадратные и сортировка**: Переписать положительные элементы массива в новый массив, преобразуя их в корни квадратные и деля на 5, затем сортировать.
